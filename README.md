# Sensor Cloud Engine

Welcome to the "Project CS 2013" project

## Running the project

0. Download and compile the dependencies, and compile the project sources
        make install

1. Run the application
        make run

## Running tests

0. Download and compile the dependencies, and compile the project sources
        make install

1. Run the tests
        make test
